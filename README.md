# TodoListBasic

A simple Java console application for managing daily tasks.

## Focus Area

- Java fundamentals
- ArrayList usage
- Methods
- User input
- Console menu design
- Basic CRUD operations

## Features

- Add tasks
- Display all tasks
- Update task details
- Delete tasks
- Search tasks by title
- Manage title, description, and time

## Purpose

This project is designed to practice Java basics through a simple task management scenario.

## Technologies

- Java
- Git
- GitHub

## How to Run

```bash
javac Main.java
java Main
```

## Learning Notes

This project is useful for practicing application flow, collections, and simple CRUD logic before moving to Spring Boot and database-backed applications.
